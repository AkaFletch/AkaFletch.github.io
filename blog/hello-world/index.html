Wing

I’m currently trying to write my own compiler and programming language called Wing. I literally started it the day of writing this post so it’s really not much. I’m hoping I keep developing it. The language is supposed to be a toy language but I’ll see how far I take it. It’s going to be strongly and statically typed using a LLVM backend to compile to multiple platforms. As for great new features I want to add to Wing, I don’t have many ideas currently, I’m just wanting to learn more about the process of compiling since previously I’ve only implemented recursive decent. The best idea I have right now is adding multiple return variables, which isn’t exactly new.

My friend Cameron made a really nice logo though. So I feel I should continue development on Wing simply to not let it goto waste. Check them out, they’re pretty cool.
